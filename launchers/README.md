# Zuno Launchers

This folder keeps the Windows launcher scripts grouped by runtime mode.

- `Zuno-Docker-*.cmd`: full Docker stack launchers
- `Zuno-Local-*.cmd`: local app launchers that reuse the existing local desktop mode

Recommended desktop shortcuts:

- Docker:
  - `Zuno-Start`
  - `Zuno-Stop`
  - `Zuno-Rebuild`
  - `Zuno-Full-Rebuild`
- Local:
  - `Zuno-Local-Start`
  - `Zuno-Local-Stop`
  - `Zuno-Local-Rebuild`
  - `Zuno-Local-Full-Rebuild`
