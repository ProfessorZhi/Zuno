# Future Architecture

## Purpose

This directory records long-term direction. It is not the current refactor
target.

Future material exists so near-term design does not block later evolution. It
must not pull the current mainline away from Python FastAPI + Service Layer +
LangGraph + RAG/GraphRAG.
