# Frontend Map

## Purpose

Orient Agents working in `apps/web`.

## Current Paths

- Vue entry: `apps/web/src/main.ts`
- Router: `apps/web/src/router/index.ts`
- Pages: `apps/web/src/pages/`
- Components: `apps/web/src/components/`
- API clients: `apps/web/src/apis/`
- Shared utilities and types: `apps/web/src/utils/`
- Design assets: `apps/web/src/assets/`

## Preferred Commands

```powershell
npm run frontend:lint
npm run frontend:build
```

## Update Triggers

Update this map when frontend entrypoints, API clients, route structure, design
system, or validation commands change.
