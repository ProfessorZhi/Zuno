# Modules

## Application Surfaces

- `apps/web/`: web application shell.
- `apps/desktop/`: desktop application shell.

## Runtime Surface

- `src/backend/zuno/`: current backend runtime truth.

## Documentation And Workflow

- `docs/`: formal documentation truth.
- `docs/architecture/`: formal architecture entrypoint.
- `.agent/programs/`: active independent Agent execution programs.
- `docs/history/`: archived and superseded plans.
- `AGENTS.md`: repository-level Agent entrypoint.
- `.agent/`: Agent workflow library.

## Support Surfaces

- `tools/`: launchers, eval tools, scripts, and maintenance commands.
- `tests/`: verification surface.
- `infra/`: infrastructure and deployment support.
