# Research Archives

这里保存研究输入、外部分析报告和 ChatGPT 研究模式产物。它们用于后续架构设计、执行计划和评估，不是当前架构事实源。

## 边界

- 当前正式架构事实源仍是 `docs/architecture/architecture.md`。
- 架构 HTML 仍由 `docs/architecture/architecture.md` 生成。
- 研究报告可以比当前实现更超前；吸收结论时必须重新区分 Current、Target、Future 和 History。
- PDF 原件优先保留，同名 Markdown 抽取版用于检索和局部引用。

## 子目录

- `chatgpt-research-mode-artifacts/`：用户提供或生成的高质量 ChatGPT 研究模式架构分析报告。
