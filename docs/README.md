1. 面向公开阅读的项目说明
2. 面向维护者的工程与架构文档

如果你第一次阅读这个仓库，优先按下面顺序看：

1. [README.md](../README.md)
2. [architecture/README.md](./architecture/README.md)
3. [development/README.md](./development/README.md)

- `architecture/` 负责稳定架构说明和 phase 计划入口
- `development/` 负责工程规则、目录治理、发布边界和维护者说明
