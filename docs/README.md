# Documentation Index

`docs/` is the stable human-facing documentation root for Zuno.

## Boundary

- Current facts, target summaries, roadmap status, formal decisions, public
  evidence, and stable reference material belong here.
- Detailed execution phases, temporary audits, Agent prompts, migration plans,
  and local operating aids belong in `.agent/` or `docs/architecture/history/`.
- Historical material is archived; it is not deleted just because it is no
  longer current.
