# Architecture Decisions

这个目录放关键架构决策记录。

它只回答一类问题：

```text
为什么最后选择了这条架构路径，
而不是另外一条看起来也能走的路径。
```

当前保留两份真正还在影响主线的决策：

- [ADR 0001: Domain Pack Binding](./0001-domain-pack-binding.md)
- [ADR 0002: Rename agentchat to zuno](./0002-rename-agentchat-to-zuno.md)

后面如果继续新增 ADR，优先记录这类决策：

- 影响长期边界的 runtime 决策
- 影响 retrieval contract 的治理决策
- 影响目录、服务边界、公开接口的结构决策
